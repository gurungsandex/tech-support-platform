# Tech Support Platform

A secure, privacy-first platform connecting end users with verified IT professionals.

## Tech Stack
- **Frontend**: Next.js 14+, React, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **Hosting**: Vercel
- **Security**: Cloudflare (DDoS, CAPTCHA)
